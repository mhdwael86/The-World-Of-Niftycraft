# The-World-Of-Niftycraft
The World of Nifty Craft is a 2D fantasy sandbox multiplayer RPG featuring a Real Time Card Battler combat system.

![Screenshot](Screenshot.jpg)

## 🕹️ About the Project
The World of Nifty Craft is a 2D fantasy sandbox multiplayer RPG featuring a Real Time Card Battler combat system.

## 🔧 Features
- MMO sandbox RPG
- Card battler
- Gathering, Crafting, Refinig, Battle
